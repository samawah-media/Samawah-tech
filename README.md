# Samawah Digital Showcase

واجهة عرض لمشاريع سماوة مع صفحة هبوط تفاعلية ولوحة أدمن لإدارة كروت المشاريع وروابط العرض المخصصة.

## التشغيل المحلي

1. ثبت الحزم:
   `npm install`
2. شغل المشروع:
   `npm run dev`
3. افتح:
   `http://localhost:3000`

## ما الذي يدعمه المشروع الآن؟

- صفحة هبوط تعرض مشاريع سماوة بصور واضحة من `public/projects`.
- لوحة أدمن على المسار `/admin`.
- إنشاء وتعديل وإخفاء وترتيب كروت المشاريع.
- رفع صورة المشروع من الجهاز مباشرة من نافذة تعديل المشروع.
- روابط عرض مخصصة مثل `/v/investors` لإظهار مجموعة مشاريع محددة لزائر معين.
- بيانات افتراضية احتياطية إذا تعذر الاتصال بـ Firebase.

## رفع صورة أفضل من الجهاز

من لوحة الأدمن:

1. افتح `/admin` وسجل دخولك.
2. اضغط تعديل على المشروع المطلوب.
3. اضغط `رفع صورة من الجهاز`.
4. اختر صورة `PNG` أو `JPG` أو `WebP`.
5. انتظر حتى يظهر رابط الصورة تلقائيا في خانة مسار الصورة.
6. اضغط حفظ.

يفضل أن تكون الصورة أفقية بنسبة قريبة من `16:9` أو `4:3`، وبحجم أقل من `2MB`، حتى تظهر بسرعة وبوضوح في الكروت.

## قواعد Firebase المطلوبة للرفع

حتى يعمل رفع الصور من لوحة الأدمن، يجب نشر قواعد `storage.rules` على Firebase Storage. القواعد تسمح للزوار بقراءة الصور، وتسمح بالرفع فقط للإيميل:

`omarsamawah@gmail.com`

ملفات القواعد الموجودة في المشروع:

- `firestore.rules`
- `storage.rules`
- `firebase.json`

إذا لم يكن Firebase CLI مسجل دخول عندك، انشر قواعد Storage من Firebase Console:

1. افتح Firebase Console.
2. ادخل مشروع `samawah-tech`.
3. افتح `Storage`.
4. افتح تبويب `Rules`.
5. انسخ محتوى `storage.rules`.
6. اضغط Publish.
