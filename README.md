# Samawah Digital Showcase

واجهة عرض لمشاريع سماوة مع صفحة هبوط تفاعلية ولوحة أدمن لإدارة الكروت وروابط العرض المخصصة.

## التشغيل المحلي

1. ثبّت الحزم:
   `npm install`
2. شغّل المشروع:
   `npm run dev`

## ما الذي يدعمه المشروع الآن؟

- صفحة هبوط تعرض المشاريع بصور محلية من مجلد `صور المشاريع`.
- لوحة أدمن على المسار `/admin`.
- روابط عرض مخصصة مثل `/v/investors` لإظهار مجموعة مشاريع محددة للزائر.
- ترتيب وإخفاء وتحرير المشاريع من واجهة الأدمن.
- إنشاء روابط مخصصة موجهة لفئات مختلفة مثل مستثمرين أو عملاء أو عروض خاصة.

## ملاحظة مهمة

إذا لم يتم ربط Firebase الحقيقي، سيعمل المشروع على بيانات تجريبية فقط. هذا يكفي للعرض والنشر الأولي، لكن تغييرات لوحة الأدمن لن تُحفَظ لجميع الزوار إلا بعد إضافة إعدادات Firebase الفعلية.
